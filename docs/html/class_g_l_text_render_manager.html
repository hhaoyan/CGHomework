Detailed Description

Manages text rendering.

A pure static class. Before any text could be rendered, you must setup the typeface(AKA font name) by calling SetFontname().

Coordinates of texts are computed using screen relative units. In horizontal direction, x starts with -1 at left and ends with +1 at right, in vertical direction, y starts with -1 at bottom and ends with +1 at top.
